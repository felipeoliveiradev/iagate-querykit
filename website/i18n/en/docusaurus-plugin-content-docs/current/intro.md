QueryKit (iagate-querykit) provides minimal, typed building blocks for SQL‑centric data apps:

- QueryBuilder to compose SQL
- Views and Triggers (SQLite)
- Scheduler
- Parallel execution
- Simulation (dry‑run)
- Multi‑database (fan‑out)
