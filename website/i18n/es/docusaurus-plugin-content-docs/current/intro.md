QueryKit (iagate-querykit) ofrece bloques de construcción mínimos y tipados para apps de datos centradas en SQL:

- QueryBuilder para componer SQL
- Vistas y Disparadores (SQLite)
- Programador (scheduler)
- Ejecución en paralelo
- Simulación (dry‑run)
- Multi‑base de datos (fan‑out)

Continúa con:
- Comenzando
- Núcleo: QueryBuilder, raw y table, Modelos
- Base de datos: Adaptadores y Ejecutores, Multi‑DB
- Vistas y Disparadores: Vistas, Disparadores SQL, Disparadores Semánticos
- Utilidades: Programador, Paralelo, Simulación
- API (TypeDoc): generada con `npm run api:docs`
